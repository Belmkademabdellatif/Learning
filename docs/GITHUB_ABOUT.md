# CodeLearn: Open Source Programming Learning Platform with AI

I've built a comprehensive interactive learning platform that combines modern web technologies with AI to create an engaging educational experience.

## 🎯 What is CodeLearn?

CodeLearn is a full-stack learning management system specifically designed for programming education. It provides:

- **Interactive Lessons**: Rich Markdown content with code examples
- **Live Code Editor**: Monaco editor (VS Code) integrated in browser
- **Secure Execution**: Docker-based sandbox for running student code
- **AI Assistant**: GPT-4 chatbot with RAG using Weaviate vector DB
- **Auto-Grading**: Instant test case evaluation
- **Certificates**: Automated PDF generation with QR verification
- **OAuth**: Google & GitHub authentication

## 🛠️ Technical Highlights

- **Frontend**: Next.js 14 with App Router, TypeScript, TailwindCSS
- **Backend**: NestJS with Prisma ORM, PostgreSQL, Redis, BullMQ
- **AI**: OpenAI GPT-4, Weaviate for semantic search
- **Security**: OWASP Top 10 compliant, JWT auth, sandboxed execution
- **Deployment**: Docker, Kubernetes, CI/CD with GitHub Actions
